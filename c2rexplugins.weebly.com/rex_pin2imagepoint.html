rex_pin2imagepoint

Plugin, ACE table, Discussion thread
Dependence: None
Introduction
Pin the position of this object to target object's image point.
Usage
Just call "Action:Pin to image point" to pin this object to other object's image point.
The different between this behavior and official pin behavior
The position of image point could be different at each frame, so that the related distance will not be fixed.
