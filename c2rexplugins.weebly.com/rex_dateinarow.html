rex_dateInARow

Plugin, ACE table, Discussion thread
Dependence: None
Introduction
Get continuous date count.
For example, date in "Day" scale -
Input date 2015/04/30 -> no continuous "Day"
Input date 2015/05/01 -> 2015/04/30 - 2015/05/01, 1 continuous "Day"
Input date 2015/05/02 -> 2015/04/30 - 2015/05/01 - 2015/05/02, 2 continuous "Day"
Input date 2015/05/04 -> 2015/05/02 - 2015/05/04 , no continuous "Day"
Usage
Sample capx
Paste
Call "Action:Paste date" to paste a new timestamp in Year/Month/Day/Hour/Minute scale at a record.
Then get continuous date count by "Expression:ContinuousCount", or the last pasted timestamp by "Expression:LastTimestamp".
Remove
Call "Action:Remove record" or "Action:Remove all" to remove one record or all records.
Save/load
This plugin supports official save/load feature.
Or get content by "Expression:AsJSON" to save states, then call "Action:Load" for loading manually.
