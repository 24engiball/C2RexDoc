rex_ptoggleswitch

Plugin, ACE table, Discussion thread
Dependence: None
Introduction
A switch for toggling between on and off.
It highlights the value changing in trigger conditions.
Usage
Sample capx
Initial value is set at "Initial value" in property table. Call
  • Action:Set value, or Action:Set value by number
  • Action:Toggle switch
to set or toggle the value.
When value changed, "Condition:On turn on" or "Condition:On turn off" will be triggered. i.e. these triggers will not be fired of value is not changed by "Action:Set value, or Action:Set value by number".
