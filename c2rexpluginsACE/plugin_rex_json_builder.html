Properties

Actions

Add value

Action:Add value

  • Add number or string value.
    • Key ( string ) : The name of the key.
    • Value ( number or string ) : The value to set, could be number or string.

Action:Add boolean value

  • Add boolean value.
    • Key ( string ) : The name of the key.
    • Boolean ( Combo ) : Boolean value.
      • Options : ["false","true"]

Action:Add null value

  • Add null value.
    • Key ( string ) : The name of the key.

Clean

Action:Clean

  • Clean all data.

Expressions

Timescale

Expression:AsJSON

  • Get JSON string.

Conditions

Add object

Condition:Add object

  • Add an object.
    • Key ( string ) : Key of this object.
    • Type ( Combo ) : Array or Dictionary.
      • Options : ["Array","Dict"]

Condition:Set root

  • Set root object.
    • Type ( Combo ) : Array or Dictionary.
      • Options : ["Array","Dict"]
