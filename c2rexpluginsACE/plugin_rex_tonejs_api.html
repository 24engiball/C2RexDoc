Properties

Transport

  • Transport for timing musical events.

Start timeline

  • Set Yes to start timeline.
    • Options : ["No","Yes"]

BPM

  • The Beats Per Minute of the Transport.

Actions

Properties

    • Properties ( string ) : Properties in JSON

Transport

Action:Set BPM

  • Set BPM.
    • BPM ( number ) : Beats Per Minute

Transport - timeline

Action:Start

  • Start timeline.
    • Time ( number or string ) : The time when the transport should start. Time in seconds(1), Notation("4n", "8t"), TransportTime("4:3:2"), Frequency("8hz"), Now-Relative("+1"), Expressions("3:0 + 2 - (1m / 7)")
    • Offset ( number or string ) : The timeline offset to start the transport. Time in seconds(1), Notation("4n", "8t"), TransportTime("4:3:2"), Frequency("8hz"), Now-Relative("+1"), Expressions("3:0 + 2 - (1m / 7)")

Action:Stop

  • Stop timeline.
    • Time ( number or string ) : The time when the transport should stop. Time in seconds(1), Notation("4n", "8t"), TransportTime("4:3:2"), Frequency("8hz"), Now-Relative("+1"), Expressions("3:0 + 2 - (1m / 7)")

Action:Pause

  • Pause timeline.
    • Time ( number or string ) : The time when the transport should pause. Time in seconds(1), Notation("4n", "8t"), TransportTime("4:3:2"), Frequency("8hz"), Now-Relative("+1"), Expressions("3:0 + 2 - (1m / 7)")
