Behavior : Shake mod

Author:Rex.Rainbow

  • Shake an object in the X and Y axis.
Download

Properties

Activated

  • Enable if you wish this to begin at the start of the layout.
    • Options : ["No","Yes"]

Duration

  • The time the shake should last, in seconds.

Magnitude

  • The strength of the shake, in pixels.

Magnitude mode

  • Decay or constant magnitude.
    • Options : ["Constant","Decay"]

Actions

Duration

Action:Set duration

  • Set duration.
    • Duration ( number ) : The time the shake should last, in seconds.

Enable

Action:Set activated

  • Enable the object's rotation behavior.
    • Activated ( Combo ) : Enable the rotation behavior.
      • Options : ["No","Yes"]

Magnitude

Action:Set magnitude

  • Set magnitude.
    • Magnitude ( number ) : The strength of the shake, in pixels.

Action:Set magnitude mode

  • Set magnitude mode.
    • Mode ( Combo ) : Decay or constant magnitude.
      • Options : ["Decay","Constant"]

Shake

Action:Start

  • Shake object for a duration of time.

Action:Stop

  • Stop shaking.

Expressions

Duration

Expression:Duration

  • Get duration.

Expression:Remainder

  • Get remaining time.

Origin

Expression:OX

  • Get X co-ordinate of shaking origin.

Expression:OY

  • Get Y co-ordinate of shaking origin.

Conditions

Shake

Condition:On shacking end

  • Triggered when shacking end.

Condition:Is shaking

  • Is object shaking.
