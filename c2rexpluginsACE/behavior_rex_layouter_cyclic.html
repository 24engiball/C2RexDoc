Behavior : Cyclic layout

Author:Rex.Rainbow

  • Put instnces cyclic on layouter.
Download

Properties

Shape

  • Shape of this circle.
    • Options : ["Circle","Ellipse"]

Mode

  • Average mode: layout instances in range averagely, Fix mode: layout instances with fix angle.
    • Options : ["Average","Fix"]

Start angle

  • Start angle of first instance, in degree.

Range angle

  • Range angle, in degree. Negative is anti-clockwise. Used in average mode.

Delta angle

  • Delta angle, in degree. Negative is anti-clockwise. Used in fix mode.

Angle offset

  • Angle offset of each sprite, in degree.

Actions

Angle

Action:Set start angle

  • Set the start angle.
    • Start ( number ) : Start angle, in degree.

Action:Add to start angle

  • Add value to start angle.
    • Delta ( number ) : Delta angle, in degree.

Average mode

Action:Set range angle

  • Set amount of range angle clockwise from start, in degrees. Negative is anti-clockwise.
    • Range angle ( number ) : Amount of Range angle clockwise from start, in degrees.

Fix mode

Action:Set delta angle

  • Set amount of delta angle clockwise from start, in degrees. Negative is anti-clockwise.
    • Delta angle ( number ) : Amount of dleta angle clockwise from start, in degrees.

Mode

Action:Set mode

  • Set mode of layout.
    • Mode ( Combo ) : Mode of layout.
      • Options : ["Average","Fix"]

Expressions

Angle

Expression:StartAngle

  • Get start angle, in degrees.
