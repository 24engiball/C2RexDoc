Plugin : Live2D Object

Author:Rex.Rainbow

  • Load and play Live2D object. http://www.live2d.com/en
Download

Properties

Color

  • Color for showing at editor.

Hotspot

  • Choose the location of the hot spot in the object.
    • Options : ["Top-left","Top","Top-right","Left","Center","Right","Bottom-left","Bottom","Bottom-right"]

Idle motion

  • Motion name of idle. Set "" if no idle motion.

Breathing

  • Enable breathing.
    • Options : ["Disable","Enable"]

Lip sync

  • Enable lip sync.
    • Options : ["Disable","Enable"]

Actions

Camera

Action:Set model scale

  • Set model scale without changing object size.
    • Scale ( number ) : Set 1 to original size.

Expression

Action:Set expression

  • Set expression.
    • Expression name ( string ) : Expression name.

Lip sync

Action:Lip sync

  • Enable or disable lip sync.
    • Enable ( Combo ) : Enable the lip sync.
      • Options : ["Disable","Enable"]

Action:Set lip sync value

  • Set lip sync value.
    • Value ( number ) : Lip sync value. Range is 0~1

Load model

Action:Load

  • Load model from model.json.
    • URI ( string ) : Enter the URL on the web, or data URI, of a model JSON to load.

Look

Action:Look at

  • Look at a specific position.
    • X ( number ) : Position X.
    • Y ( number ) : Position Y.

Action:Look front

  • Look front.

Motion

Action:Start motion

  • Start motion.
    • Motion name ( string ) : Motion name.

Action:Set idle motion

  • Set idle motion.
    • Motion name ( string ) : Motion name.

Parameter

Action:Set value

  • Set the value of model's parameter.
    • Parameter name ( string ) : Parameter name.
    • Value ( number ) : Value.

Action:Add to

  • Add to the value of a model's parameter.
    • Parameter name ( string ) : Parameter name.
    • Value ( number ) : Value.

Part

Action:Set part opacity

  • Set part opacity.
    • Part index ( string ) : Part index.
    • Opacity ( number ) : Opacity, range from 0 to 1.

breathing

Action:Breathing

  • Enable or disable Breathing.
    • Enable ( Combo ) : Enable the breathing behavior.
      • Options : ["Disable","Enable"]

Expressions

Model - load

Expression:LoadedFailedFilePaths

  • Get path of loaded failed files. Joins by ";".

Model scale

Expression:ModelScale

  • Get current model scale. 1 is original size

Motion

Expression:MotionName

  • The name of the current motion.

Expression:TriggeredMotionName

  • The name of the triggered motion, used under "Condition: On any motion began", or "Condition: On any motion finished"

Expression:MotionData

  • Get data of current motion. Add 1st parameter to get value at the specific key. Add 2nd parameter for default value if this key is not existed.

Conditions

Hit test

Condition:Is inside area

  • Return true if the position is inside the area.
    • X ( number ) : Position X.
    • Y ( number ) : Position Y.
    • Area name ( string ) : Enter the name of the area to check.

Model

Condition:Is ready

  • Retrun true if model is ready to use.

Model - load

Condition:On loaded successful

  • Triggered when model is loaded successful.

Condition:On loaded failed

  • Triggered when model is loaded failed.

Motion

Condition:Is motion playing

  • Test which of the object's motion is currently playing.
    • Motion name ( string ) : Enter the name of the motion to check if playing.

Motion - began

Condition:On motion began

  • Triggered when a motion has began.
    • Motion name ( string ) : Enter the name of the motion to check if playing.

Condition:On any motion began

  • Triggered when any motion has began.

Motion - finished

Condition:On motion finished

  • Triggered when a motion has finished.
    • Motion name ( string ) : Enter the name of the motion to check if playing.

Condition:On any motion finished

  • Triggered when any motion has finished.
