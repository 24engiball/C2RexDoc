Plugin : UserID to ID

Author:Rex.Rainbow

  • Mapping between UserID and another alias ID.
Download

Properties

Domain

  • The root location of the Firebase data.

Sub domain

  • Sub domain for this function.

Actions

Request - ID

Action:Get random ID

  • Request random ID.
    • User ID ( string ) : Key of User ID.
    • Digits ( number ) : Count of digit characters for default value.
    • Retry ( number ) : Retry count to get default value.

Action:Try set ID

  • Request try set ID.
    • User ID ( string ) : Key of User ID.
    • ID ( string ) : ID.

Action:Get ID

  • Request ID.
    • User ID ( string ) : Key of User ID.

Request - User ID

Action:Get User ID

  • Request User ID.
    • ID ( string ) : ID.

Expressions

Request - result

Expression:ID

  • Get ID of result.

Expression:UserID

  • Get User ID of result.

Conditions

Request - ID

Condition:On request ID successfully

  • Triggered when requested ID successfully.

Condition:On request ID failed

  • Triggered when requested ID failed.

Request - User ID

Condition:On request User ID successfully

  • Triggered when requested User ID successfully.

Condition:On request User ID failed

  • Triggered when requested User ID failed.
