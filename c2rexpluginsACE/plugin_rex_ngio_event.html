Plugin : Event

Author:Rex.Rainbow

  • Handles logging of custom events.
Download

Properties

Actions

Event

Action:Log event

  • Logs a custom event to your API stats.
    • Event name ( string ) : The name of your custom event as defined in your Referrals & Events settings.
    • Host ( string ) : The domain hosting your app. Example: "newgrounds.com", "localHost"

Expressions

Log

Expression:EventName

  • Get logged event name from last result.

Result

Expression:ErrorMessage

  • Get last error message from last result.

Conditions

Log

Condition:On log event

  • Triggered when log event success.

Condition:On log event error

  • Triggered when log event error.
