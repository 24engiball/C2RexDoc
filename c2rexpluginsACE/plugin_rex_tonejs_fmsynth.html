Plugin : FMSynth

Author:Rex.Rainbow

  • Composed of two Tone.Synths where one Tone.Synth modulates the frequency of a second Tone.Synth.
Download

Properties

Actions

Configuration

Action:Set portamento

  • Set portamento.
    • Portamento ( number or string ) : The glide time between notes

Action:Set detune

  • Set detune.
    • Detune ( number ) : The glide time between notes

Action:Set harmonicity

  • Set harmonicity.
    • Harmonicity ( number ) : Harmonicity is the ratio between the two voices. A harmonicity of 1 is no change. Harmonicity = 2 means a change of an octave.

Action:Set modulation index

  • Set modulation index.
    • ModulationIndex ( number ) : The modulation index which essentially the depth or amount of the modulation. It is the ratio of the frequency of the modulating signal (mf) to the amplitude of the modulating signal (ma) -- as in ma/mf.

Creator

Action:Create

  • Create instrument.
    • Options ( string ) : Options in JSON string.

Envelope

Action:Set envelope

  • Set envelope.
    • Attack ( number ) : When triggerAttack is called, the attack time is the amount of time it takes for the envelope to reach it's maximum value.
    • Decay ( number ) : After the attack portion of the envelope, the value will fall over the duration of the decay time to it's sustain value.
    • Sustain ( number ) : The sustain value is the value which the envelope rests at after triggerAttack is called, but before triggerRelease is invoked.
    • Release ( number ) : After triggerRelease is called, the envelope's value will fall to it's miminum value over the duration of the release time.

Filter envelope

Action:Set filter envelope

  • Set filter envelope.
    • Attack ( number ) : When triggerAttack is called, the attack time is the amount of time it takes for the envelope to reach it's maximum value.
    • Decay ( number ) : After the attack portion of the envelope, the value will fall over the duration of the decay time to it's sustain value.
    • Sustain ( number ) : The sustain value is the value which the envelope rests at after triggerAttack is called, but before triggerRelease is invoked.
    • Release ( number ) : After triggerRelease is called, the envelope's value will fall to it's miminum value over the duration of the release time.
    • Base frequency ( number or string ) : The envelope's mininum output value. This is the value which it starts at.
    • Octaves ( number ) : The number of octaves above the baseFrequency that the envelope will scale to.
    • Exponent ( number ) : The envelope's exponent value.

Oscillator

Action:Set type

  • Set oscillator type.
    • Prefix ( Combo ) : Prefix of type.
      • Options : ["","am","Left to right","fat"]
    • Type ( Combo ) : Scan direction.
      • Options : ["sine","square","triangle","custom","pwm","pulse"]

Oscillator - custom

Action:Set partials

  • Set partials, only if the oscillator is set to "custom".
    • Partials ( string ) : Partials in an array.

Oscillator - pulse

Action:Set width

  • Set width, only if the oscillator is set to "pulse".
    • Width ( number ) : The width of the oscillator.

Trigger

Action:Attack then release

  • Trigger the attack and then the release after the duration.
    • Note ( number or string ) : The note to trigger. Note-Octave("C4") or frequency(262).
    • Duration ( number or string ) : How long the note should be held for before triggering the release. Time in seconds(1), Notation("4n", "8t"), TransportTime("4:3:2"), Frequency("8hz"), Now-Relative("+1"), Expressions("3:0 + 2 - (1m / 7)")
    • Time ( number or string ) : When the note should be triggered. Time in seconds(1), Notation("4n", "8t"), TransportTime("4:3:2"), Frequency("8hz"), Now-Relative("+1"), Expressions("3:0 + 2 - (1m / 7)")
    • Velocity ( number ) : The velocity the note should be triggered at, within the range [0, 1]

Action:Attack

  • Trigger the attack of the note optionally with a given velocity.
    • Note ( number or string ) : The note to trigger. Note-Octave("C4") or frequency(262).
    • Time ( number or string ) : When the note should be triggered. Time in seconds(1), Notation("4n", "8t"), TransportTime("4:3:2"), Frequency("8hz"), Now-Relative("+1"), Expressions("3:0 + 2 - (1m / 7)")
    • Velocity ( number ) : The velocity the note should be triggered at, within the range [0, 1]

Action:Release

  • Trigger the release portion of the envelope.
    • Time ( number or string ) : When the note should be triggered. Time in seconds(1), Notation("4n", "8t"), TransportTime("4:3:2"), Frequency("8hz"), Now-Relative("+1"), Expressions("3:0 + 2 - (1m / 7)")

Action:Set note

  • Set the note at the given time.
    • Note ( number or string ) : The note to trigger. Note-Octave("C4") or frequency(262).
    • Time ( number or string ) : When the note should be triggered. Time in seconds(1), Notation("4n", "8t"), TransportTime("4:3:2"), Frequency("8hz"), Now-Relative("+1"), Expressions("3:0 + 2 - (1m / 7)")
