Behavior : Text resize

Author:Rex.Rainbow

  • Resize text object and background to show all lines.

Properties

Auto resize

  • Enable to resize object automatically.
    • Options : ["No","Yes"]

Actions

Background

Action:Add background

  • Add background object.
    • Baclground ( object ) : Baclground object.
    • Resize ( Combo ) : Resize mode
      • Options : ["width and height","heigh only"]

Resize

Action:Resize

  • Resize text object to show all text with minimal height.

Action:Set max width

  • Set max width of this object.

Expressions

Conditions

Resize

Condition:On size changed

  • Triggered when size changed.
