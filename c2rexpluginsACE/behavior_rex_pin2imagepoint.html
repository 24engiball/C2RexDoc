Behavior : Pin to image point

Author:Rex.Rainbow

  • Stick to another object's image point.
Download

Properties

Actions

Pin

Action:Pin to image point

  • Pin the object to another object's image point.
    • Pin to ( object ) : Choose the object to pin to.
    • ImagePoint ( number or string ) : Name or number of image point to get.
    • Mode ( Combo ) : Choose which properties of the object to update.
      • Options : ["Position only","Position & angle"]

Unpin

Action:Unpin

  • Unpin the object.

Expressions

Expression:PinnedUID

  • Get the UID of the object pinned to, or -1 if not pinned.

Conditions

Condition:Is pinned

  • Object is currently pinned to another object.
