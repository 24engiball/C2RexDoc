Behavior : Anchor mod

Author:Rex.Rainbow

  • Position objects relative to the size of the window.

Properties

Left edge

  • Anchor the object's left edge to a window edge.
    • Options : ["Window left","Window right","None"]

Top edge

  • Anchor the object's top edge to a window edge.
    • Options : ["Window top","Window bottom","None"]

Right edge

  • Anchor the object's right edge.
    • Options : ["None","Window right"]

Bottom edge

  • Anchor the object's bottom edge.
    • Options : ["None","Window bottom"]

Initial state

  • Whether to initially have the behavior enabled or disabled.
    • Options : ["Disabled","Enabled"]

Set once

  • Enable to set position once when window size changed.
    • Options : ["No","Yes"]

Actions

Anchor

Action:Set enabled

  • Enable or disable the behavior.
    • State ( Combo ) : Whether to enable or disable the behavior.
      • Options : ["disabled","enabled"]

Expressions

Conditions

Anchor

Condition:On anchored

  • Triggered when anchored.
