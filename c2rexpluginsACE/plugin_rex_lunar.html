Plugin : Lunar

Author:Rex.Rainbow

  • Transfer date from solar to lunar (solar 1900~ 2049)
Download

Properties

Actions

Lunar

Action:Solar to lunar

  • Transfer date from solar to lunar.
    • Year ( number ) : Year (1900~2049)
    • Month ( number ) : Month (1~12)
    • Day ( number ) : Day (1~31)

Expressions

Lunar

Expression:Year

  • Get year value.

Expression:Month

  • Get month value.

Expression:Day

  • Get day value.

Expression:IsLeap

  • Get leap value, return 1 if leap.

Expression:Gan

  • Get gan value.

Expression:Zhi

  • Get zhi value.

Expression:ShengXiao

  • Get sheng-xiao value.

Conditions
