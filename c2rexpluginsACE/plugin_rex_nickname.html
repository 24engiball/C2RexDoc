Plugin : Nickname

Author:Rex.Rainbow

  • Create object by nickname.
Download

Properties

Actions

Assign

Action:Assign nickname

  • Assign nickname.
    • Nickname ( string ) : Nickname of object.
    • Object ( object ) : Object type.

Create

Action:Create instance

  • Create instance by nickname.
    • Nickname ( string ) : Nickname of object.
    • X ( number ) : X co-ordinate.
    • Y ( number ) : Y co-ordinate.
    • Layer ( layer ) : Layer name of number.

Action:Create instance into family

  • Create instance by nicknamem then put into family.
    • Nickname ( string ) : Nickname of object.
    • X ( number ) : X co-ordinate.
    • Y ( number ) : Y co-ordinate.
    • Layer ( layer ) : Layer name of number.
    • Family ( object ) : Family object for SOL.

SOL

Action:Pick all instances

  • Pick all.
    • Nickname ( string ) : Nickname of object.
    • Family ( object ) : Family object for SOL.

Action:Pick matched instances

  • Pick matched instances.
    • Substring ( string ) : Substring to get matched nicknames.
    • Family ( object ) : Family type.

Expressions

Created instance

Expression:LastCreatedInstUID

  • Get last created instance UID.
