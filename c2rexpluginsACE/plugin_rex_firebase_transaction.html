Plugin : Transaction

Author:Rex.Rainbow

  • Read-modify-write the value.
Download

Properties

Actions

Read

Action:Transaction

  • Transaction
    • DataRef ( string ) : The Firebase data ref URL

Write

Action:Return value

  • Returns transaction value.
    • Value ( number or string ) : The value to set

Action:Return JSON

  • Returns transaction JSON.
    • JSON value ( string ) : JSON value to set

Action:Return null

  • Returns transaction value to null. i.e. remove data.
    • Value ( number or string ) : The value to set

Action:Return boolean

  • Returns transaction value to true or false.
    • Boolean ( Combo ) : Boolean value.
      • Options : ["false","true"]

Action:Abort

  • Abort transaction.

Write - JSON

Action:Return value

  • Set value in returned JSON.
    • Key ( number or string ) : The name of the key.
    • Value ( number or string ) : The value to set, could be number or string.

Action:Return boolean

  • Set boolean in returned JSON.
    • Key ( number or string ) : The name of the key.
    • Boolean ( Combo ) : Boolean value.
      • Options : ["false","true"]

Expressions

Read

Expression:ValueIn

  • Read value, using under "condition:On transaction". Add 1st parameter for specific key, 2nd parameter for default value if this key is not existed.

Result

Expression:ValueOut

  • Transaction wrote result, using under "condition:On completed". Add 1st parameter for specific key, 2nd parameter for default value if this key is not existed.

Conditions

Read

Condition:ValueIn is null

  • Return true if ValueIn is null.

Result

Condition:Is aborted

  • Return true if the transaction is aborted.

Condition:On complete

  • Triggered while transaction success.

Condition:On error

  • Triggered while transaction error.

Condition:ValueOut is null

  • Return true if ValueOut is null.

Write

Condition:On transaction

  • Triggered by calling "action: Transaction", to get return value.
