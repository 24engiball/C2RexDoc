Properties

Actions

Connect

Action:Connect

  • Connect to object.
    • Object ( object ) : Object to plug.
    • Port ( string ) : Port name. set "" to ignore this parameter.

Control

Action:Start

  • Start the source at the specified time.
    • Parameter {n} ( variadic ) : Parameters of this function call.

Action:Stop

  • Stop the source at the specified time.
    • Parameter {n} ( variadic ) : Parameters of this function call.

Creator

Action:Create oscillator

  • Create oscillator.
    • Oscillator type ( string ) : Oscillator type.
    • Parameter {n} ( variadic ) : Parameters of this function call.

Dispose

Action:Dispose

  • Clean up.

Properties

Action:Set JSON

  • Set properties to JSON string.
    • Properties ( string ) : Properties in JSON

Property

Action:Set value

  • Set property.
    • Property ( string ) : Property name in dot notation
    • Value ( number or string ) : Value to set

Action:Set JSON

  • Set property to JSON string.
    • Property ( string ) : Property name in dot notation
    • JSON ( string ) : JSON value to set

Action:Set boolean

  • Set property to a boolean value.
    • Property ( string ) : Property name in dot notation
    • Boolean ( Combo ) : Boolean value.
      • Options : ["false","true"]

Sync

Action:Sync

  • Sync the source to the Transport so that all subsequent calls to start and stop are synced to the TransportTime instead of the AudioContext time.

Action:Unsync

  • Unsync the source to the Transport.

Action:Sync frequency

  • Sync the signal to the Transport's bpm. Any changes to the transports bpm, will also affect the oscillators frequency.

Action:Unsync frequency

  • Unsync the oscillator's frequency from the Transport.
