Behavior : Rotate

Author:Rex.Rainbow

  • Rotate Sprite
Download

Properties

Activated

  • Enable if you wish this to begin at the start of the layout.
    • Options : ["No","Yes"]

Direction

  • Select clockwise or anticlockwise rotation.
    • Options : ["Anti-clockwise","Clockwise"]

Speed

  • Rotation speed, in degrees per second.

Acceleration

  • Rotation acceleration, in degrees per second per second, negative slows down.

Actions

Action:Set activated

  • Enable the object's rotation behavior.
    • Activated ( Combo ) : Enable the rotation behavior.
      • Options : ["No","Yes"]

Action:Set rotation direction

  • Set the object's rotation direction.
    • Direction ( Combo ) : Select clockwise or anticlockwise rotation.
      • Options : ["Anti-clockwise","Clockwise"]

Action:Set rotation speed

  • Set the object's current rotation speed.
    • Rotation speed ( number ) : Rotation speed, in degrees per second.

Action:Set rotation acceleration

  • Set the object's rotation acceleration.
    • Rotation acceleration ( number ) : The acceleration setting, in pixels per second per second.

Expressions

Expression:Speed

  • The current object rotation speed, in degrees per second.

Expression:Acceleration

  • The rotation acceleration setting, in degrees per second per second.

Expression:Activated

  • The activated setting, 1 is activated.

Conditions

Condition:Compare rotation speed

  • Compare the current rotation speed of the object.
    • Comparison ( cmp ) : Choose the way to compare the current rotation speed.
    • Rotation speed ( number ) : The rotation speed, in degrees per second, to compare the current speed to.
