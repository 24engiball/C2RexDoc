Plugin : Layouter

Author:Rex.Rainbow

  • Layout instances to get position and angle.
Download

Properties

Pin mode

  • Moving instances like pin behavior.
    • Options : ["None","Position & angle","Position only","Angle only"]

Hotspot

  • Choose the location of the hot spot in the object.
    • Options : ["Top-left","Top","Top-right","Left","Center","Right","Bottom-left","Bottom","Bottom-right"]

Actions

Add/Remove

Action:Add instances

  • Add instances.
    • Instances ( object ) : Add instances into layouter.

Action:Create&Add instances

  • Create and Add instances.
    • Instances ( object ) : Add instances into layouter.
    • X ( number ) : X co-ordinate.
    • Y ( number ) : Y co-ordinate.
    • Layer ( layer ) : Layer name of number.

Action:Remove instances

  • Remove instances.
    • Instances ( object ) : Remove instances from layouter.

Action:Remove all

  • Remove all instances.

SOL: instances

Action:Pick instances

  • Pick instances.
    • Object ( object ) : Object for picking

Action:Pick all instances

  • Pick all instances in this layouter.

Update

Action:Force layout

  • Force layout instances.

Expressions

Layout

Expression:InstUID

  • Get UID of layouted instance.

Expression:InstX

  • Get X co-ordinate of layouted instance.

Expression:InstY

  • Get Y co-ordinate of layouted instance.

Expression:InstAngle

  • Get angle of layouted instance.

Expression:InstWidth

  • Get width of layouted instance.

Expression:InstHeight

  • Get height of layouted instance.

Expression:InstOpacity

  • Get opacity of layouted instance.

Expression:InstVisible

  • Get visible of layouted instance.

Sprite

Expression:SpritesCnt

  • Get sprites count.

Conditions

Layout

Condition:On layout instance

  • Trigger by "Action:Add instances" or "Action:Create&Add instances" to layout instance.

SOL: instances

Condition:Pick instances

  • Pick instances.
    • Object ( object ) : Object for picking

Condition:Pick all instances

  • Pick all instances in this layouter.

SOL: layouter

Condition:Pick layouter

  • Pick layouter by instance.
    • Instance ( object ) : Instance under layouter
