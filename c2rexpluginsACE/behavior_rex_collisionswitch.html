Behavior : Collision switch

Author:Rex.Rainbow

  • Enable or disable collision setting.

Properties

Collisions

  • Whether the object will register collision events or not.
    • Options : ["Disabled","Enabled"]

Actions

Collision

Action:Set collisions enabled

  • Set whether the object will register collision events or not.
    • Collisions ( Combo ) : Whether to enable or disable collisions for this object.
      • Options : ["Disabled","Enabled"]

Expressions

Conditions

Collisions

Condition:Collisions enabled

  • True if the object's collisions are enabled and will fire collision events.
