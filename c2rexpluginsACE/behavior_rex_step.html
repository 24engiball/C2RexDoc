Behavior : Step

Author:Rex.Rainbow

  • Insert x,y position.
Download

Properties

Activated

  • Enable if you wish this to begin at the start of the layout.
    • Options : ["No","Yes"]

Stepping mode

  • Move in increments while firing the step triggers.
    • Options : ["Linear","Horizontal then vertical","Vertical then horizontal"]

Pixels per step

  • When stepping enabled, the number of pixels to move each step.

Noise shift

  • Noise shift in pixel per step.

Actions

Action:Set activated

  • Enable the object's step behavior.
    • Activated ( Combo ) : Enable the behavior.
      • Options : ["No","Yes"]

Action:Set pixels per step

  • Set pixels per step.
    • Pixels per step ( number ) : The pixels per step.

Action:Force stepping

  • Force stepping.

Velocity

Action:Stop stepping

  • Stop the current stepping, preventing any more step triggers firing this tick.
    • Position ( Combo ) : Choose the resulting object position.
      • Options : ["Go back a step","Stay at current position"]

Expressions

Conditions

Condition:On step

  • Triggered when stepping the object for motion.

Condition:On horizontal step

  • Triggered when stepping the object horizontally for motion.

Condition:On vertical step

  • Triggered when stepping the object vertically for motion.
