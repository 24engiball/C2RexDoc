Behavior : Logic Pin

Author:Rex.Rainbow

  • Change logic position (X, Y) with pined chess.

Properties

Activated

  • Enable if you wish this to begin at the start of the layout.
    • Options : ["No","Yes"]

Actions

Enable

Action:Set activated

  • Enable this behavior.
    • Activated ( Combo ) : Enable the behavior.
      • Options : ["No","Yes"]

Pin

Action:Pin to chess

  • Pin the chess to another chess.
    • Pin to ( object ) : Choose the chess to pin to.

Action:Pin to chess by UID

  • Pin the chess to another chess by UID.
    • UID of chess ( number ) : Choose the chess to pin to.

Unpin

Action:Unpin

  • Unpin the chess.

Expressions

Conditions

Logical position

Condition:On logical position changed

  • Triggered when logical position changed.

Pin

Condition:Is pinned

  • Object is currently pinned to another object.
