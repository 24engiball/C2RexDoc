Behavior : Angle lock

Author:Rex.Rainbow

  • Lock the sprite angle.

Properties

Activated

  • Enable if you wish this to begin at the start of the layout.
    • Options : ["No","Yes"]

Angle

  • The angle of sprite, in degree.

Actions

Angle

Action:Set locked angle

  • Set locked angle, in degree.
    • Angle ( number ) : The locked angle, in degree.

Enable

Action:Set enabled

  • Enable or disable the behavior.
    • State ( Combo ) : Whether to enable or disable the behavior.
      • Options : ["disabled","enabled"]
