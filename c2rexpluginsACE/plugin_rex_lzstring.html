Plugin : Lz-string

Author:Rex.Rainbow

  • Compress string using LZ-based compression algorithm. Reference - https://github.com/pieroxy/lz-string

Properties

Actions

Local

Action:Set local value

  • Store a value in local storage (available in any session).
    • Key ( string ) : Enter the name of the key to associate the value with.
    • Value ( number or string ) : Enter the value to store.

Expressions

Compress

Expression:Compress

  • Compress source string.
    • Source ( string ) : Source string to compress

Decompress

Expression:Decompress

  • Decompress result string.
    • Result ( string ) : Result string to decompress

Local

Expression:LocalValue

  • Get the value from a key in local storage. Return the default value if the key does not existed, and save the default value to webstorage.
    • Key ( string ) : Key name
    • Default ( number or string ) : The default value if the key does not existed.

Conditions
