Plugin : Container

Author:Rex.Rainbow

  • Put sprites or other objects into this box.
Download

Properties

Pin mode

  • Moving instances like pin behavior.
    • Options : ["None","Position & angle","Position only","Angle only"]

Hotspot

  • Choose the location of the hot spot in the object.
    • Options : ["Top-left","Top","Top-right","Left","Center","Right","Bottom-left","Bottom","Bottom-right"]

Tag

  • Tag for adding sprite into container at layout editor.

Actions

Add instances

Action:Add instances

  • Add instances.
    • Instances ( object ) : Add instances into container.

Action:Create&Add instances

  • Create and Add instances.
    • Instances ( object ) : Add instances into container.
    • X ( number ) : X co-ordinate.
    • Y ( number ) : Y co-ordinate.
    • Layer ( layer ) : Layer name of number.

Remove instances

Action:Remove instances

  • Remove instances.
    • Instances ( object ) : Remove instances from container.

SOL: instances

Action:Pick instances

  • Pick instances.
    • Object ( object ) : Object for picking

Action:Pick all instances

  • Pick all instances in this container.

Size & Position

Action:Set scale

  • Set the width and height as a multiple of its original size.
    • Scale ( number ) : The object width and height to set, based on a multiple of its original dimensions, e.g. 1 = original size, 2 = double size, 0.5 = half size etc.

Expressions

Tag

Expression:Tag

  • Get tag.

Conditions

SOL: container

Condition:Pick container

  • Pick container from instance.
    • Instance ( object ) : Instance under container

SOL: instances

Condition:Pick instances

  • Pick instances.
    • Object ( object ) : Object for picking

Condition:Pick all instances

  • Pick all instances in this container.
