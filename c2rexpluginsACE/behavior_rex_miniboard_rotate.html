Behavior : (Miniboard) Rotate

Author:Rex.Rainbow

  • Spin chess on mini board logically and physically.
Download

Properties

Direction

  • Face direction.

RotateTo

  • Set Yes to change the logical and physical position of chess. Set No will only change the logical position.
    • Options : ["No","Yes"]

Actions

Face

Action:Face

  • Rotate mini board to face to direction.
    • Direction ( number ) : Face direction.
    • Putable test ( Combo ) : Mode of putable test.
      • Options : ["None","Empty","Putable"]

Action:Face

  • Rotate mini board to face to direction.
    • Direction ( Combo ) : Face direction.
      • Options : ["Right","Down-right","Down-left","Left","Up-left","Up-right"]
    • Putable test ( Combo ) : Mode of putable test.
      • Options : ["None","Empty","Putable"]

Flip

      • Options : ["None","Empty","Putable"]

Rotate

  • Spin chess on mini board.
    • Direction ( number ) : Rotating direction.
    • Putable test ( Combo ) : Mode of putable test.
      • Options : ["None","Empty","Putable"]

Action:Rotate
