Behavior : Push Out Solid

Author:Rex.Rainbow

  • Push out from solid object.
Download

Properties

Activated

  • Enable if you wish this to begin at the start of the layout.
    • Options : ["No","Yes"]

Obstacles

  • Choose whether solids obstruct or if to use custom objects added by events.
    • Options : ["Solids","Custom"]

Actions

Enable

Action:Set enabled

  • Set whether this behavior is enabled.
    • State ( Combo ) : Set whether to enable or disable the behavior.
      • Options : ["Disabled","Enabled"]

Obstacle

Action:Add obstacle

  • Add a custom object as an obstacle to line-of-sight.
    • Obstacle ( object ) : Choose an object to add as an obstacle, obstructing line-of-sight.

Action:Clear obstacles

  • Remove all added obstacle objects.

Push out

Action:Push out nearest

  • Push out nearest.
    • Max push distance ( number ) : Max push distance, in pixels.

Action:Push out at angle

  • Push out at angle.
    • Angle ( number ) : Angle, in degrees, to push the object out at.
    • Max push distance ( number ) : Max push distance, in pixels.

Action:Push out toward position

  • Push out toward position.
    • X ( number ) : The X co-ordinate in the layout to accelerate toward.
    • Y ( number ) : The Y co-ordinate in the layout to accelerate toward.
