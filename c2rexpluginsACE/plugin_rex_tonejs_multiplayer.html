Plugin : MultiPlayer

Author:Rex.Rainbow

  • Play a bunch of audio buffers.
Download

Properties

Actions

Control

Action:Start

  • Start a buffer by name.
    • Buffer name ( string ) : The name of the buffer to start.
    • Time ( number or string ) : When to start the buffer. Set "" to ignore this parameter.
    • Offset ( number or string ) : The offset into the buffer to play from. Set "" to ignore this parameter.
    • Duration ( number or string ) : How long to play the buffer for. Set "" to play full length of the sample (minus any offset).
    • Pitch ( number or string ) : The interval to repitch the buffer. Set "" to play full length of the sample (minus any offset).
    • Gain ( number or string ) : The gain to play the sample at. Set "" to play full length of the sample (minus any offset).

Action:Stop

  • Stop the buffer at the specified time.
    • Buffer name ( string ) : The name of the buffer to start.
    • Time ( number or string ) : When to stop the buffer. Set "" to ignore this parameter.

Action:Stop all

  • Stop all currently playing buffers at the given time.
    • Time ( number or string ) : When to stop the buffers. Set "" to ignore this parameter.

Action:Start loop

  • Start a looping buffer by name.
    • Buffer name ( string ) : The name of the buffer to start.
    • Time ( number or string ) : When to start the buffer. Set "" to ignore this parameter.
    • Offset ( number or string ) : The offset into the buffer to play from. Set "" to ignore this parameter.
    • Loop start ( number or string ) : The loop start time.
    • Loop end ( number or string ) : The loop end time.
    • Pitch ( number or string ) : The interval to repitch the buffer. Set "" to play full length of the sample (minus any offset).
    • Gain ( number or string ) : The gain to play the sample at. Set "" to play full length of the sample (minus any offset).

Creator

Action:Create player

  • Create player.
    • URL ( string ) : name:url pairs .

Dispose

Action:Dispose

  • Clean up.

Load

Action:Add

  • Add another buffer to the available buffers.
    • Buffer name ( string ) : The name of the buffer to start.
    • URL ( string ) : The url from which to load the AudioBuffer.

Properties

Action:Set JSON

  • Set properties to JSON string.
    • Properties ( string ) : Properties in JSON

Property

Action:Set value

  • Set property.
    • Property ( string ) : Property name in dot notation
    • Value ( number or string ) : Value to set

Action:Set JSON

  • Set property to JSON string.
    • Property ( string ) : Property name in dot notation
    • JSON ( string ) : JSON value to set

Action:Set boolean

  • Set property to a boolean value.
    • Property ( string ) : Property name in dot notation
    • Boolean ( Combo ) : Boolean value.
      • Options : ["false","true"]

Sync

Action:Sync

  • Sync the source to the Transport so that all subsequent calls to start and stop are synced to the TransportTime instead of the AudioContext time.

Action:Unsync

  • Unsync the source to the Transport.

Conditions
