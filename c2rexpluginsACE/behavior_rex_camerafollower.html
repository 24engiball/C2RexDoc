Behavior : Camera follower

Author:Rex.Rainbow

  • Move object to follow camera with a ratio. It could be used to simulate parallax.
Download

Properties

Type

  • Object type.
    • Options : ["Follower","Camera"]

Activated

  • Enable if you wish follower followers camera.
    • Options : ["No","Yes"]

X

  • Moving ratio of position X of follower.

Y

  • Moving ratio of position Y of follower.

Actions

Follower

Action:Set following

  • Enable or disable following.
    • State ( Combo ) : Enable or disable following.
      • Options : ["Disabled","Enabled"]

Mving ratio

Action:Set moving ratio X

  • Set moving ratio of position X.
    • X ( number ) : Moving ratio of position X.

Action:Set moving ratio Y

  • Set moving ratio of position Y.
    • Y ( number ) : Moving ratio of position Y.

Expressions

Mving ratio

Expression:MovingRatioX

  • Get moving ratio of position X.

Expression:MovingRatioY

  • Get moving ratio of position Y.

Conditions
